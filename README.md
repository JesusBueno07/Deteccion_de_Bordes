# Deteccion de Bordes
Este repositorio contiene implementaciones básicas de distintos algoritmos de detección de bordes aplicados a imágenes. Incluye filtros clásicos como Roberts, Prewitt, Sobel y Canny, así como etapas de procesamiento como conversión a escala de grises y suavizado previo.

El objetivo es explorar y comparar diferentes métodos de detección de bordes mediante implementaciones sencillas en Python, facilitando el análisis de cómo cada filtro resalta cambios de intensidad en una imagen.
